  <name>Task 1: Add most_downloads sort entry and update installed comparator in sort.lua</name>
  <files>lua/store/sort.lua</files>
  <action>
In `lua/store/sort.lua`:

1. Update the `installed` sort comparator to use the context table pattern:
   - Change `fn = function(a, b, installed_items)` to `fn = function(a, b, ctx)`
   - Extract installed_items from ctx: `local installed_items = ctx and ctx.installed_items`
   - Rest of the function body stays the same (already guards `installed_items and installed_items[a.name]`)

2. Add a new `most_downloads` entry to `M.sorts` (place it AFTER `installed`):
   ```lua
   most_downloads = {
     label = "Most Downloads",
     fn = function(a, b, ctx)
       if not ctx or not ctx.download_stats then
         return false -- no-op: preserve current order when stats not loaded
       end
       local a_count = ctx.download_stats[a.full_name] or 0
       local b_count = ctx.download_stats[b.full_name] or 0
       return a_count > b_count
     end,
   },
   ```

3. Update `get_sort_types()` to append `"most_downloads"` at the end of the returned array.

Note: The other sorts (most_stars, recently_updated, recently_created) use `_` for the 3rd param so they need NO changes.
